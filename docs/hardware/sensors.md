#### 4-in-1 Soil Sensor (Zigbee)

A multi-function sensor that measures soil moisture, air temperature, air humidity, and luminance.

##### Features
- Soil moisture detection
- Air temperature detection
- Air humidity detection
- Luminance detection

##### Compatibility
- **Requires:** A Zigbee 3.0 gateway running Zigbee2MQTT.

##### Technical Specifications
- **Wireless Protocol:** Zigbee 3.0
- **Power:** 2 x LR03 AAA batteries
- **Working Voltage:** DC 3V
- **Standby Current:** ≤20uA
- **Measurement Resolution:**
    - **Soil Moisture:** 3%
    - **Temperature:** 0.1℃
    - **Humidity:** 3%
    - **Light Intensity:** 1 lux
- **Measurement Accuracy:**
    - **Temperature:** ±0.5℃
    - **Humidity:** ±5%
